# e-Nazir MyClaim System 🚀

An enterprise-grade **Claim Management & Budget Monitoring System** built with **Django**. This platform centralizes financial claims and provides a "Single Source of Truth" for real-time budget tracking across multiple sectors.

---

### 🌟 Key Capabilities
* **Granular RBAC (Role-Based Access Control):** Custom dashboards and permissions tailored for KNS (Monitoring), PTDM (Planning), Finance, and Administrative roles.
* **Real-time Analytics:** Integrated **Chart.js** for interactive visual reporting of monthly expenditure trends and sector-wise budget distribution.
* **Automated Approval Workflow:** Streamlined claim lifecycle—from submission to approval (Sah) or rejection with audit-ready ulasan (comments).
* **Live Budget Safeguard:** Built-in validation to prevent over-claiming by comparing real-time expenditure against allocated program estimates.
* **Modern Interface:** Responsive **Glassmorphism UI** built with Bootstrap 5, featuring a dynamic sidebar and mobile-friendly layouts.

---

### 🛠️ Tech Stack & Architecture
* **Core Engine:** Python 3.x / Django (Robust Backend)
* **Frontend:** JavaScript (ES6+), HTML5, CSS3 (Modern Crystal UI)
* **Libraries:** Chart.js (Data Visualization), DataTables.net (Advanced Search/Filtering)
* **Database:** SQLite (Development) / PostgreSQL-ready
* **Authentication:** Django Identity Management (Secure Login/Access)

---

### 💡 The Problem It Solves
Traditional claim processes are often fragmented and prone to human error. **e-Nazir MyClaim** automates these workflows, significantly reducing manual data entry and providing management with high-level visibility into financial health, ensuring budget compliance at every stage.
